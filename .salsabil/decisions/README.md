# Decision Log (ADRs)

Append-only. Never edit a past ADR — supersede it with a new one.

## Numbering

`NNNN-kebab-title.md`, monotonic, zero-padded to 4 digits.

## Required sections

See `_template.md`. Every ADR MUST include: Status, Date, Sponsor, Domain, Context, Decision, Alternatives, Consequences, Migration plan, Observability impact.
