# Reef Al Madina (ريف المدينة)

Primary retail super-app of Salsabil OS. Houses storefront, supermarket, meat, pharmacy, recipes, sweets, library, baskets, group-buy, POS, vendor, driver, account, and the main-hub launcher.

All features here consume the OS kernel under `@/core-os/*`:
- **Tayseer** (`@/core/finance`) — wallet, balance, savings, charity
- **Hakim AI** (`@/core/hakim-ai`) — central intelligence brain
- **SDUI Engine** (`@/core/runtime-ui/sdui`) — server-driven layouts
- **Barq Logistics** (`@/core/logistics`) — geo + delivery (formerly Mrsool/Rawafid)
- **System Editor** (`@/core/runtime-ui/system-editor`) — visual layout editor
